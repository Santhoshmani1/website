2 · Get this online

You're previewing a file straight from your device — that only works in this browser tab, right now, so there's no embed code to generate yet. To turn this into something you can actually paste into your site:

1. Upload this mp3 to a static host — GitHub Pages, Cloudflare Pages, Netlify, Vercel, or an S3 bucket with public-read + CORS all work well.
2. Copy the public URL it gives you.
3. Switch to the Link tab above, paste that URL in, and hit Load preview again — the embed code will appear.

On hosting the mp3: the browser needs to fetch the actual audio bytes from wherever you host it, and — because this visualizer reads the raw waveform via the Web Audio API — that host must serve the file with permissive CORS headers (Access-Control-Allow-Origin: *), or the bars will simply stay flat while audio still plays. GitHub Pages, Cloudflare R2/Pages, Netlify, Vercel, and most S3 buckets with public-read + CORS enabled all work well for this. A plain link to a file on Google Drive or Dropbox generally will not work — export the audio through a static host or CDN instead.

This whole page is one HTML file with inline CSS/JS and no dependencies besides the two Google Fonts links — save it, host it anywhere, and the iframe above will keep working.
